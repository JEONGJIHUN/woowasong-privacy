우와송 고객 지원

우와송은 아이의 목소리로 음악을 만드는 앱입니다. 사용 중 궁금한 점이 있으시면 아래 자주 묻는 질문을 먼저 확인해 주세요.

자주 묻는 질문

마이크 권한이 왜 필요한가요?
우와송은 아이의 목소리를 듣고 실시간으로 음정을 변환해 들려주는 앱입니다. 그래서 마이크 접근 권한이 꼭 필요합니다. 마이크로 입력된 소리는 기기 안에서만 처리되며, 어떤 서버로도 전송되지 않습니다.
녹음 파일은 어디에 저장되나요?
모든 녹음은 사용자의 기기 안에만 저장됩니다. 부모 영역에서 녹음을 재생, 저장, 삭제할 수 있어요. 별표를 누르지 않은 녹음은 최근 항목 중심으로 자동 정리될 수 있습니다.
소리가 안 나거나 변환이 이상해요.
다음 순서로 확인해 보세요.
1) 기기 음량과 무음 모드 스위치 확인
2) 설정 → 우와송 → 마이크 권한이 켜져 있는지 확인
3) 다른 앱(전화, 음성 녹음 등)이 마이크를 사용 중인지 확인 후 종료
4) 앱을 완전히 종료 후 다시 실행
그래도 해결되지 않으면 아래 문의 채널로 알려 주세요.
아이가 사용해도 안전한가요?
우와송은 아이가 직접 사용하는 것을 전제로 만들어졌습니다. 계정 가입을 요구하지 않고, 광고나 외부 추적 기능도 사용하지 않습니다. 위치, 연락처, 사진 보관함, 웹 업로드 같은 민감한 권한도 사용하지 않습니다.
화면이 가로로만 보여요. 세로로 쓸 수 있나요?
우와송은 아이가 양손으로 잡고 캐릭터 버튼을 누르는 환경에 맞춰 가로 화면으로 고정되어 있습니다. 세로 모드는 지원하지 않습니다.
저장한 녹음을 다른 곳으로 옮길 수 있나요?
부모 영역에서 녹음을 재생하고 관리할 수 있습니다. 외부 공유 기능은 향후 업데이트로 추가될 수 있으며, 변경 사항은 이 페이지와 처리방침 페이지에서 안내합니다.
